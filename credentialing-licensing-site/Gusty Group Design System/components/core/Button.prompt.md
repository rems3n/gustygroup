**Button** — flat, square, 1px-bordered action that inverts on hover. Use for every CTA; solid `primary`/`olive` for the main action, `ghost` variants for secondary, `on-dark` on olive surfaces.

```jsx
<Button variant="olive" arrow href="#book">Book a 30-minute call</Button>
<Button variant="ghost-ink">See if we're a fit</Button>
```

Variants: `primary` (solid ink), `olive` (solid green), `ghost` (olive outline), `ghost-ink` (ink outline), `on-dark` (paper outline for olive backgrounds). Sizes: `sm` (nav), `md` (default). Set `arrow` for the ↗ booking affordance.
